What is AniMalSync?

AniMalSync is a project I created when I first switched to using AniList over MyAnimeList. Having used MyAnimeList for several years, however, I didn't want to completely drop the site in case I ever decided to go back to it. Manually updating both sites was a hassle, so I created a command line script to automate the process.

Originally, the script ran on my Raspberry Pi, but once I got familiar with AWS tools, I decided to port it to a webapp. Every 25 minutes, the app uses the AniList GraphQL to fetch updates from your profile and reflect those changes to your MyAnimeList account using MAL's REST API.

AniMalSync is an unofficial app and is not officially associated with either AniList or MyAnimeList in any capacity.
